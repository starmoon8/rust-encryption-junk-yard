ascon

Strong Ascon-128 file encryptor for Linux

v2.0.0 • Linux only

A simple, fast, and security-first CLI tool that encrypts individual files using Ascon-128 (NIST Lightweight Cryptography winner) with an extremely strong Argon2id key derivation (512 MiB memory hardness).

Features

  • Ascon-128 AEAD (authenticated encryption)
  • Aggressive Argon2id defaults (512 MiB, 10 iterations, 4 lanes)
  • Password confirmation on encrypt
  • --keyfile support for non-interactive scripts and hardware tokens
  • Always produces .ascon files (original extension stored inside)
  • Atomic writes + secure permissions (0600)
  • --shred option for secure deletion
  • info command
  • Sensitive data zeroized from memory

Usage Examples

# Normal password mode
ascon encrypt readme.md
ascon decrypt readme.ascon
ascon info readme.ascon

# Encrypt and securely shred original
ascon encrypt --shred important.txt

Keyfile mode (non-interactive / hardware tokens)

--keyfile (or -k) is a flag you add to encrypt or decrypt.

The keyfile must be exactly 16 bytes of raw binary data.

Why exactly 16 bytes? Ascon-128 (the official NIST standard we use) is designed with 128-bit keys. This is intentional and correct — longer keys are not supported in the Ascon-128 AEAD specification.

How to create a keyfile

# Recommended (uses OpenSSL)
openssl rand -out KEY.bin 16

# Alternative (using Linux random)
dd if=/dev/random of=KEY.bin bs=16 count=1 status=none

Examples with keyfile

# Encrypt without typing a password
ascon encrypt --keyfile KEY.bin secret.pdf

# Decrypt using the same keyfile
ascon decrypt --keyfile KEY.bin secret.pdf.ascon

# Combine with shred
ascon encrypt --keyfile KEY.bin --shred important.txt
Important notes:
• Keep your keyfile extremely secure — anyone who has it can decrypt your files.
• No independent security audit has been performed yet.
• Loads the entire file into RAM (not suitable for huge files).
• Linux only.
• For full-disk encryption use LUKS or gocryptfs.

Installation

cargo install --git https://github.com/YOURNAME/ascon.git
